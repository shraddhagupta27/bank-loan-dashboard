# Bank Loan Performance Dashboard (Tableau)

### Project Overview
An interactive Tableau dashboard that visualizes key performance indicators for a bankâ€™s loan portfolio. The dashboard tracks lending activity, repayment trends, and borrower profiles to support data-driven decision-making in financial analysis and risk management.

### Objectives
- Monitor total loan applications, funded amount, and repayments over time  
- Track Month-to-Date (MTD) and Month-over-Month (MoM) changes in key KPIs  
- Segment loans by purpose, term, employment length, and home ownership
- Identify regional lending patterns across U.S. states  
 

### Tools & Technologies
- Tableau for dashboard design  
- Excel for data preprocessing  
- Calculated Fields for MTD, MoM, and KPI logic
